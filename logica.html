Ejercicio: Control de Temperaturas

Control de Temperaturas

Programa en Java que permite registrar temperaturas diarias en un arreglo y mostrar estadísticas básicas.

Código Java - ControlTemperatura.java 

import java.util.Scanner;

public class ControlTemperatura {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);

    double[] temperaturas = new double[7];
    int contador = 0;
    String respuesta = "s";

    System.out.println("Bienvenido al Control de Temperaturas Semanal");

    while (respuesta.equalsIgnoreCase("s") && contador < 7) {
      try{
        System.out.print("Ingrese la temperatura del día " + (contador + 1) + ": ");
        temperaturas[contador] = input.nextDouble();
        contador++;

        if (contador < 7) {
          System.out.print("¿Deseas ingresar otra temperatura? (s/n): ");
          respuesta = input.next();

        } else {
          System.out.println(" Ya ingresaste las 7 temperaturas de la semana.");
          break;
        }
      } catch (Exception e) {
        System.out.println("\nHubo un error al procesar los datos, vuelve a intentar\n");
        input.nextLine();
      }

    }

    System.out.println("\n== Temperaturas registradas: ==");
    for (int i = 0; i < contador; i++) {
      System.out.println("Día " + (i + 1) + ": " + temperaturas[i] + "°C");
    }
    double max = temperaturas[0];
    double min = temperaturas[0];

    for (int i = 1; i < contador; i++) {
      if (temperaturas[i] > max) {
        max = temperaturas[i];
      }
      if (temperaturas[i] < min) {
        min = temperaturas[i];
      }
    }

    System.out.println("\nTemperatura más alta: " + max + "°C");
    System.out.println("Temperatura más baja: " + min + "°C");


    System.out.println("\nGracias por usar el Control de Temperaturas");
  }
}
Características
  • Arreglo de 7 posiciones para temperaturas
  • Ciclo while para entrada de datos
  • Validación de entrada del usuario
  • Manejo de excepciones
Funcionalidades
  • Registro de temperaturas diarias
  • Muestra todas las temperaturas
  • Calcula temperatura máxima
  • Calcula temperatura mínima
